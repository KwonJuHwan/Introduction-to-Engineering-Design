디지털 탄소발자국을 줄이기 위한 방법

당신의 탄소 발생량을 줄이기 위한 방법입니다.

  • 동영상 시청으로 발생하는 탄소 발생량을 줄이고 싶다면?

    1.동영상의 해상도를 낮춰주세요. 해상도를 낮추는것 만으로도 탄소 발생량이 줄어듭니다.

    2.스트리밍 대신 다운로드 해주세요. 자주 보는 동영상은 매번 스트리밍 하는것보다 다운로드 후에 보는것이 좋습니다.

    3.동영상 자동 재생을 차단하세요. 자신이 보고있지 않거나 자고 있을때 자동으로 실행되는 동영상도 탄소 발생을 일으킵니다.

  • 전화 통화로 인한 탄소 발생량을 줄이고 싶다면??

    1.통화 시간을 줄여주세요. 통화시 용건만 간단히, 짧게 하는것이 환경 보호에 도움이 됩니다.

    2.영상통화 시간을 줄여주세요. 영상통화는 추가적으로 데이터를 사용하기에 탄소를 더욱 발생시키게 됩니다.

  • 데이터 사용으로 인한 탄소 발생량을 줄이고 싶다면?

    1.데이터는 쓰지 않을때 꺼주세요. 데이터가 필요한 순간이 아니라도 데이터를 켜놓으면 데이터가 사용될 때가 있기 때문에 데이터를 꺼놓으면 데이터 절약에 도움이 되고, 환경에도 도움이 됩니다.

    2.데이터 요금제를 바꿔주세요. 예를들어, 한달 10G 데이터 요금제보단 2G 요금제를 사용하면 돈도 절약되고 탄소도 절약됩니다.

  • 이메일로 인한 탄소 발생량을 줄이고 싶다면?

    1.불필요한 광고 수신이나 뉴스레터 구독을 끊어주세요. 사용하지 않는데도 수신되는 각종 메일들도 탄소 발생량을 높이게 됩니다.

    2.필요없는 메일들은 전부 지워주세요. 메일함에 쌓여있는 메일들 또한 탄소 발생을 일으키게 됩니다. 메일을 지우는 것 만으로도 환경에 보호가 됩니다.

  • 마지막으로

    스마트폰으로 인한 탄소 배출을 줄이는 가장 좋은 방법은 스마트폰 사용 시간을 줄이는 것입니다. 자신이 스마트폰을 평균보다 많은 시간동안 사용하고 있다면 스마트폰 사용 시간을 줄임으로써 스마트폰 중독도 예방하고 환경 보호에도 이바지 할 수 있는 일석이조가 됩니다. 오늘부터 스마트폰 사용 시간을 줄여보고 다음에 다시 이 홈페이지의 디지털 탄소발자국 계산기를 사용하여 자신이 얼마나 환경 보호에 도움이 되었는지 계산해보세요!
